## 📖 О проекте

`SuperApp` — это проект, демонстрирующий создание модульного десктопного приложения с использованием Python, современного графического интерфейса и работы с данными. Приложение состоит из главного окна с навигацией и набора утилит.

## 💡 Использование
Шаг 1: Установите Python

Шаг 2: Создайте и активируйте виртуальное окружение
(Выполните команду:)

``` bash
python -m venv .venv
```
(Активируйте окружение:)

Windows: `.venv\Scripts\activate`

Linux/macOS: `source .venv/bin/activate`

Шаг 3: В терминале перейдите в корневую папку проекта (SuperAppProject1).

``` bash
cd SuperAppProject1
```

Шаг 4: Установите все необходимые библиотеки из файла requirements.txt.

``` bash
pip install -r requirements.txt
```

Шаг 5: Запустите приложение

Находясь в корневой папке проекта (SuperAppProject1), выполните команду:

``` bash
python -m SuperApp.main
```

## 🧠Статус проекта

Утилита №1: "Курсы валют" — Реализована. Позволяет отслеживать динамику стоимости выбранных валют по данным ЦБ РФ за указанный период с построением интерактивного графика.

Утилиты №2: "Бюджет и накопления" - Реализована. Утилита представляет собой комплексный финансовый менеджер, предназначенный для учета доходов и расходов, постановки финансовых целей и анализа трат.

Утилита №3: "Трекер привычек" — Реализована. Позволяет пользователям планировать ежедневные действия, следить за прогрессом и получать наглядную обратную связь в виде статистических данных.

Утилита №4: "Расписание" - Реализована. Утилита создаваёт расписание занятий с защитой от конфликтов по времени, контролировать нагрузку и получать уведомления о приближающихся парах.

Утилита №5: "Статистика игры" - Реализована. Позволяет вести личный дневник игрового опыта: ставить оценки играм, оставлять развернутые отзывы, отмечать выполненные достижения и анализировать своё игровое портфолио с помощью инфографики.

## 🛠️ Технологии
- **Python 3.13**
- UI Фреймворк: PyQt6
1) Преимущества: Высокая производительность, нативный вид на всех платформах (Windows, macOS, Linux), простота интеграции мультимедиа и веб-элементов.
2) Ключевые компоненты: QWebEngineView для встраивания интерактивных веб-компонентов, QCalendarWidget для удобного планирования, QTableWidget для сложных таблиц.

- Система управления зависимостями: pip (26.1.2) — официальный менеджер пакетов Python. Позволяет легко управлять внешними библиотеками, обеспечивая воспроизводимость среды разработки.

*- Пакеты для работы с данными и аналитикой*
1) pandas (3.0.3): Мощная библиотека для обработки и анализа табличных данных. Упрощает работу с финансовыми показателями и бюджетом.
2) NumPy (2.4.6): Основание для Pandas. Позволяет выполнять быстрые векторные вычисления и матричные операции.
3) requests (2.34.2): Простой HTTP-клиент для взаимодействия с внешними API (например, для курсов валют).

*- Визуализация и графика*
1) Plotly (6.8.0): библиотека для создания интерактивных графиков. Отлично интегрируется с PyQt6 через QWebEngineView.
2) Matplotlib (3.10.9): классический инструмент для построения статистических диаграмм. Используется для круговой диаграммы (pie-chart) в модуле "Статистика игр", позволяя наглядно представить распределение оценок.

*- Вспомогательные пакеты*
pyqt6-qt6 (6.11.0), pyqt6-sip (13.11.1), pyqt6-webengine (6.11.0), pyqt6-webengine-qt6 (6.11.1): служебные пакеты, обеспечивающие работоспособность PyQt6 и поддержку современных браузеров внутри приложения.

*- Для работы с данными и графиками:*
- requests: Для получения данных с сайта ЦБ РФ.
- pandas: Для обработки и анализа данных.
- plotly: Для создания интерактивных графиков.
